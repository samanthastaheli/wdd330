Chapter 10 Notes: Testing and Debugging

Error Types

System error: problem is with system or external device that is interacting with your program

Programming error: program has incorrect syntax or logic

User error: user enteres data inncorectly which the program cannot use

Exceptions

Error that returns a messge about the error so you can fix it.

Stack Trace

Works backwards to find error.

Strict Mode

Makes more exeptions and warnings.

Write 'use strict'; in a line of code.

Can be used inside function.

Modules are in scrict mode by default.

Linting Tools

JS Lint, JS Hint, and ES Lint

Highlight sloppy code, syntax errors, and if certain style conventions (indent in code) are not followed.

Feature Detection

Modernizer or Can I Use?

Debugging in Browser

Inspect in browser, break points

Alert

Use alert to make breakpoint. Stops program until OK is clicked.

Console

console.log() logs values/variables in the console

console.trace() show functions until reaches an exeption

Debugger

write debugger; in line of code

Creates a breakpoint. This pauses code and you can see where the program is up to.

Error Objects

create using constructor functions

const error = new Error('An error has occured');

properties: name, message, stack

Other Error Objects

  1. EvalError
  2. RangeError: when number is outside of allowed range
  3. ReferenceError: when reffrence is made to an item that does not exist
  4. SyntaxError
  5. TypeError: type of value used is wrong
  6. URIError
  7. InternalError

Throw

Inset a negative number to see throw work.

Try, Catch, Finally

Test a Function

Example:

function squareRootTest() {
return squareRoot(25) === 5;
}
console.log(squareRootTest())

Will return a boolean in the console. If true, function works.

TDD

aka test-driven development

  1. Write test that initally fail
  2. Write code to pass the test
  3. Refractor code
    • Refractor means to restrcture function but don't alter functionality
  4. Test the refractored code
  5. Write more tests for new additions/features

Jest

A TDD framework.

Facebook created Jest.

Instal by writing npm install -g jest in JavaScript code.

npm = node package manager
