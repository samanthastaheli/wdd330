Object Oriented Programming

Models objects in real world.

3 Main OOP Concepts:

Encapulation

Essential functions are seen, but rest is hidden in from user in code.

Polymorohism

Objects share same method and can override them with more implemntion.

Inheritence

This means that the object inherits properties from a previous object.

Classes

Go with constructor functions.

Use keyword class

Template for an object.

Constructor Property

functionName.constuctor

Returns constructor function that created it.

Make Copy of a Function:

  1. Make new function with new operator.
  2. Use newFuncName.constructor
  3. Change the parameter.

Example:

const greenDice = new redDice.constructor(10);
