localStorage

localStorage Definition:

Local Storage is a property that accesses a storage object for the documents orgin.

Documents orgin is were website page is from. It has a scheme (https) and a host (example.com).

localStorage vs. sessionStorage:

Session storage is cleared when the pages is closed, while local storage saves data in the browser.

localStorage Example:

localStorage.setItem(key, value);

key = object name

value = object value

key-value pairs:

A Key-value pair is a string or property name that correlates with a value (which can be anything).

Examples:

key = name

value = phone number and address

key = song title

value = song

Mobile UX Design

Rules:

  1. Remeber where your thumbs are.
    • 75% of user interactions done with thumb
    • Keep most wanted interaction in green zone.
    • Minimize targer size for accuracy because thumbs are big. Good size = 7mm or 44px or 2.75rem
  2. Have empathy for users.
    • Launch-Glance-Go-Progressive disclosure
      • don't make user scan
      • let user ask for it as needed
  3. Don't hude stuff from user.
    • controls below content
  4. Get out of users way.
    • no splash screens
    • minimal branding
