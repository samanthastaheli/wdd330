localStorage Definition:

Local Storage is a property that accesses a storage object for the documents orgin.

Documents orgin is were website page is from. It has a scheme (https) and a host (example.com).

localStorage vs. sessionStorage:

Session storage is cleared when the pages is closed, while local storage saves data in the browser.

localStorage Example:

localStorage.setItem(key, value);

key = object name

value = object value

key-value pairs:

A Key-value pair is a string or property name that correlates with a value (which can be anything).

Examples:

key = name

value = phone number and address

key = song title

value = song
