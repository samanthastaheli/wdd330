Chapter 8 Notes: Transforms and Transitions

Transforms

Values of transform property separated by spaces.

Don't work on inline elements.

Have to change to inline-block to work.

Translation

Value of the transform property.

Move elements left, right, up, and down.

Uses x and y axis.

Have no impact on document flow.

If moved text around it won't move with it.

Functions

  • translate(x, y)
    • parameters are x and y values/positions
  • translatex(x)
    • parameter is x value
    • moves horizontally
  • translatey(y)
    • parameter is y value
    • moves vertically

Example

transform: translate(50px, 100px);

  • scale(x,y)
    • parameter is value

Example

transform: scale(1); // stay same size
transform: scale(2); // double size
transform: scale(0.5); // half size
transform: scale(1.5, 0.25); // resize x to half size bigger, resize y to 1/4 size

Example

use translate and scale together

executes in order written

transform: translatex(45px), scale(1.5);

  • rotate()
    • rotates around point of origin
    • default point of origin is the center

Example

transform: rotate(10deg);

  • skew(x, y)
  • skewx(x)
  • skewy(y)
    • skews x and y axis values
    • can make look like titled to the side

transform-origin

Property that manipulate elements point of origin.

Example

transform-orgin: 50% 50%; // default value
transform-orgin: 0 0;
transform-orgin: top center;

Transitions

Transitionable properties are ones that have a change in midpoint.

  • width is transitonable
  • border-style is not
    • because changing line from solid to dashed does not change the midpoint
  • exception: visible and hidden

Use keyword all to set all at once.

Steps to create transition with only CSS:

  1. Declare original state of element in the elements style declaration.
  2. Declare final state in the hover style declaration of that element.
  3. Use the transition functions in the original style declaration.

Properties

  • transition-property
  • transition-duration
    • in s or ms
      • 0.2s
      • 200ms
  • transition-timing-function
    • specify transitions start and end times
    • keywords:
      • ease: slow start then speeds up then slows at end
      • linear: constant speed
      • ease-in: start slow, end fast
      • ease-out: start fast, end slow
      • ease-in-out: start is exhilarates faster than regular ease
    • transition-delay

    cubic-bezier

    linear = cubic-bezier(0, 0, 1, 1)
