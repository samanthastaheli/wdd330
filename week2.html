Variables

const vs. let

const = use when variable will not be reassigned another value

let = use if variable will/might reassigned a different value

Immutable vs. Mutable

Immutable: An object that cannot be modified after it has been created.

Mutable: An object that can be modified after it is created.

assign multiple variables:

let x=1, y=2, z=3

reassign value:

variableName = 5

Naming Variables

Can start with lowercase, uppercase, underscore, or $.

If starts with underscore it is usually for private properties or methods.

Starting with $ is used in jQuery.

Scope

Global Scope:

  • Variables outside of block (loops, if, switches, etc.)
  • Accessable everywhere

Local Scope:

  • Variables created inside block using let or const.
  • Only accesible inside that block.

How to overwrite global variables from block?

Don't use let in the block. Without it will refer to the global variable outside the block.

Example:

let b = 2;

{ b = 4; }

How to create a global variable inside a block?

Don't use let or const when make new variable

Example:

{ x = 6; }

Converting string to number

  1. Number method: Number('27');
  2. Multiply numerical string by 1: const x = '2' * 1;
  3. + operator inf front of numberical string: const x = +'7';
  4. parseInt()
    • If string starts with number it will display only the numbers not any words.
    • If decimal displays nothing after decimal.
  5. parseFloat()
    • Converst string into floating point number.

Converting numbers to strings

  1. String function: string(3);
  2. Add empty string: 3 + ' ';
  3. toString() method

Arrays

Change length:

arrayName.length = 5;

Remove entity:

  1. pop() - removes last element
  2. shift() - removes first element
  3. splice() - removes specific elements
    • example: .splice( 0, 2 ) removes 2 elements starting at the 0 index

Add entity:

  1. push() - add 1 or more elements to end
  2. unshift() - add 1 or more elements to begining
  3. splice( 6, 0, 'g', 'h' ) - removes 0 elements from index 6 and adds elements g and h
  4. Set elements at next index position in that array.

Replacing entity:

  1. reassign entity
  2. splice( 6, 2, 's', 'o' ) - removes 2 entities at index 6 and adds s and o

Destructung arrays:

Take values out of array and assign them to variables.

Example:

let [x, y] = [1, 2];

Merge arrays:

  1. concat() method
  2. spread operator: 3 dots in front of array
    • ex: arrayName = [...arrayName, ...[1, 2, 3, 4]];
  3. join() method
    • Creates a new string of all the elements in the array
    • Seperated by commas.
    • Change seperation symbol:
      • arrayName.join('&') - now seperated by &
