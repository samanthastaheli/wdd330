Chapter 11 Notes: Further Functions

Length

All functions have a length property.

Call

Call method sets value of this inside a function.

If not using this make first argument null

squareRoot.call( null ,  4 );

.call( thisAsArgument ,  otherArgument );

Apply

Like call but argument is array.

.apply( null , [4] );

Description

Property that allows you to describe what function does.

Cache

Property that stores values of a function so don't have to do function again.

Immediately Invoked Function Expressions

  • aka IIFE ("iffy")
  • Has parentheses at end of function to invoke it.
  • Function is in parentheses to make it an expression.
  • Variables contained in scope.
  • Global namespace not polluted with other variables.

Temporary Variables

temp

Undeclared after function is invoked.

Define and Rewrite

Call the function then redefine it by rewriting it inside the function.

Promises

For resolve use .then

For reject use .catch
