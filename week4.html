Chapter 8 Notes

A Form Needs

  • Form tag needs name and action attribute
    • name is the name of the form
    • action is URL that form will be submitted to
  • input tag needs a name
  • button tag needs type

Form Controls/Tags

  1. input: text, numbers, check boxes, radio buttons, and file uploads
  2. select: makes drop-down list of options
  3. textarea: makes longer text entry
  4. button
  5. required: add the word required into input tag and input is now required

Input Types

  1. text
  2. password
    • the input will not able to be seen
  3. checkbox
    • have checked property to determine if checked
    • add word: checked inside input tag to automatically check the box
  4. radio button
    • type="radio"
    • only allow 1 option to be checked
  5. hidden
    • not seen in browser
    • seen in HTML so not for sensitive data
  6. file input
    • type="file"
  7. number
    • has arrows
  8. other types:
    • tel
    • color
    • email

How to pre-fill an input?

Use value attribute.

value="10" means that the value in that input is 10 until user changes it
